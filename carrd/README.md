# carrd

These are all code snippets for Carrd where some custom code was needed to implement features or adjust styling.

## live-translate

Implements Google Translate on the page. It is invoked with a JS function call:

```javascript
translatePage('es')
```

## seo

Adds JSON-LD object to the site metadata in order to steer search engines.

## styling-platform-items

Arranges all the platform items using flexbox layout.

## styling-letter-to-editor

Styles letter to editor in the Media section.
